# example-repo
## Triathlon Time Calculator

### Description

This Python progran calculates an athlete's total completion time (in minutes) across three categories (swimming, running and cycling).
The program determines the award the athlete will receive based on the overall completion time. 

### How the progrm works

1. User input:
   - The user is asked to enter thier completion time for all 3 events in minutes
2. The program:
   - Adds all three times together and displays the total completion time
3. Award:
   - An award gets assigned based on the total completion time
